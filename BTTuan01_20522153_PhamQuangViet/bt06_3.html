Two Column Flexible With Header

Ver 2.3

This is the same as the 2 Column Flexible but the addition of a
header div for top navigation of banner.

The left column uses float left and has a percentage width allowing both
columns to 'flow' as the browser window changes shape.

This template has been tested in:

  • Mozilla 1.5b & Firefox 0.8
  • Internet Explorer 6
  • Netscape 7.1
  • Opera 7.11
