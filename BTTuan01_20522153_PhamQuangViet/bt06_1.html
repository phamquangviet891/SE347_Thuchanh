Two Column Flexible

Ver 2.3

This is a very easy page layout which you have probably seen on sites
such as ALA (A List Apart). Each column is created with simply two div
tags.

The left column uses float left and has a percentage width allowing both
columns to 'flow' as the browser window changes shape.

This template has been tested in:

  • Mozilla 1.5b & Firefox 0.8
  • Internet Explorer 6
  • Netscape 7.1
  • Opera 7.11

Changes in this version:

  1. Defined zero for the body to fix a problem in Opera.
